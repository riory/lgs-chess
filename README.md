# LGS-Chess

## Overview
Simple chess server. Written mainly using llm.

This project is a Node.js web application built with **TypeScript 5.x**, **Express.js**, and **EJS templates**. It enforces strict coding standards using **ESLint + Prettier** (Airbnb style guide) and includes:
- TypeScript strict mode configuration
- EJS template rendering for dynamic frontend
- Mocha/Chai test suite for unit & integration testing
- SSL certificate generation scripts
- VS Code workspace settings for optimal development

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm or yarn
- Git

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/riory/lgs-chess.git
   cd lgs-chess
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm start
   ```

### Key Commands
- **Format code:** `npm run format`
- **Lint code:** `npm run lint`
- **Run unit tests:** `npm test`
